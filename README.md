Available Features :

- Anti-Link
- Autorole
- Auto-Mod
- Custom Prefix
- Permanent Database
- Welcomer
- Logging
- Menu(s)
- Slash Commands
- Rep System

Command Categories :

- Configuration
- Fun
- Image
- Info
- Moderation
- Music
- NSFW
- Utilities

Requirements :
- Discord.js v13 (npm install discord.js@latest)
- applications.commands scope enabled for your bot in Developer Portal (For Slash Cmds).
- NodeJS v16.6 or higher
- Basic knowledge of JS or Discord.JS

How To Use :
- edit      : config.json
- start bot : (node .) or ( node index.js)
